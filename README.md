# webscrap
webscraping valor dolar banco galicia, banco supervielle, banco nacion

Unas pequeñas líneas de código en Python que permiten obtener de la web de cada uno de los bancos la cotización del momento.
Está escrito de tal modo que Selenium no tiene que abrir físicamente un explorador para obtener los datos.
Se le da un tiempo de sleep para que Javascript de la página pueda cargar y no tener errores.
